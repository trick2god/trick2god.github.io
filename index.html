Core Capabilities

What NAM actually does.

Not just a monitoring tool. NAM is an execution engine — it doesn't just see problems, it fixes them.

01
🗄️
Centralized Network Device Inventory

Auto-updating database of every network device — always current, always searchable. NAM parses show commands from Cisco switches and routers to build a living database with location, version, interfaces, descriptions, and more.

02
⚡
Centralized VIP Inventory

One table showing every Virtual IP across all load balancers — backends, certificates, content switching, HA status — joined via VLOOKUP-style formulas into a single AG-Grid interface anyone can use without CLI knowledge.

03
🔔
Intelligent Incident Notifications

NAM knows what's important. When an incident ticket arrives, it filters the noise, extracts the critical fields, and pushes a clean, actionable alert to the right MS Teams group chat instantly — with full update tracking.

04
🔐
Mass Password Changes in Minutes

Annual password rotation to hundreds of Cisco and Digi devices across all sites — executed simultaneously with real-time validation output per device. What used to take days of manual work takes minutes with full auditability. Results are emailed automatically.

05
📋
Automated Certificate Renewal Review

NAM monitors Change Tasks in ServiceNow for certificate renewal activities on load balancer VIPs. It automatically reviews the installer's work, validates the certificate bindings, and produces a detailed compliance report without manual checking.

06
🌊
Ticket Flood Control

When a site is down, NAM detects it and prevents a cascade of duplicate alerts from flooding the ticket queue — one alert per event, not dozens. Intelligent suppression with full site-awareness.

07
🏥
Automated Device Healthchecks

NAM runs baseline comparisons on Cisco and Citrix devices, sends the diff in a single email showing exactly what changed in show commands — and distinguishes between real outages and false positives automatically.

08
🗺️
Device & Server Locator

Enter an IP. NAM queries PaloAlto for the MAC address, cross-references Cisco ARP tables, and returns the exact switch, port, and location the device is physically connected to. Across your entire network — in seconds.

09
🚀
Scheduled Implementations at Scale

Push ACL changes, config updates, or any network command to hundreds of devices across all sites — simultaneously, on a schedule, with per-device validation email output. Weekend maintenance windows that used to require a team are now a single NAM job.

10
📊
Live AG-Grid Databases with Formulas

NAM's front-end is powered by AG-Grid — an Excel-like interface with live data, VLOOKUP-style cross-database joins, custom formula columns, filtering, sorting, and export. Users get the power of a spreadsheet over live network data — no SQL needed.

11
📉
Utilization Charts from Live APIs

CPU, memory, disk, and network charts pulled live from Dynatrace and WhatsUpGold — rendered directly in NAM using Chart.js, so teams can troubleshoot without navigating to external tools.

12
🔄
Configuration Change Monitoring

NAM continuously monitors Cisco and Citrix devices for unauthorized or untracked configuration changes — alerting the right people via email with the full diff so nothing slips through unnoticed.

13
🔍
Filtered Report Delivery

Users save database filter views in NAM. NAM then sends those filtered results — daily, weekly, or monthly — to email or MS Teams automatically. Critical monitor lists, backup status, certificate expiry — delivered on schedule.

🤖 AI-Powered
✨
AI Prompts for Data Summarization

NAM integrates AI prompts to generate intelligent summaries of reports, healthcheck outputs, and bulk data — turning raw network telemetry into readable, actionable narratives for teams and leadership without manual write-ups.

Custom Database Engine

A query engine built for
network teams.

NAM's core API supports a ServiceNow-style query language so any team member can filter, sort, and join data from any connected system — without SQL or Python knowledge.

  • ✓ ServiceNow-style sysparm_query syntax (LIKE, IN, BETWEEN, ISEMPTY, regex, date ranges)
  • ✓ VLOOKUP-style cross-database joins — connect inventories with one formula
  • ✓ Runtime formula columns — CONCAT, IF, CHECK2TEXT, CHECKSAMETEXT, MATCH
  • ✓ Interface name normalization (expand GigabitEthernet ↔ shorten Gi)
  • ✓ Streaming JSON with ijson for handling massive datasets without memory issues
  • ✓ Topological formula dependency resolution — no circular dependency errors
  • ✓ Filter ID system — save any complex filter, reuse it in dashboards, emails, or Teams
Real-World Use Cases

Problems solved.
Every day.

These aren't hypothetical benefits. These are live automations running right now.

🚨
Site-Down Detection & Alert
Execution: Real-time

NAM detects a site is unreachable via WhatsUpGold, suppresses all child device alerts to prevent ticket flooding, and sends a single, clean MS Teams notification to the on-call team with all relevant context — before anyone even notices something is wrong.

🔑
Annual Password Rotation
Execution: Minutes, not days

Select hundreds of Cisco routers, switches, and Digi devices from the NAM inventory. Click run. NAM connects to all of them simultaneously, changes the password, validates the new credential, and emails a per-device success/failure report — all without a single manual SSH session.

📜
Certificate Renewal Audit
Execution: Automated on ticket

When a Change Task for VIP certificate renewal arrives in ServiceNow, NAM automatically reviews the installer's work — verifying that the right cert is bound to the right VIP on the right load balancer — and posts the result back to the ticket.

📡
Who Changed That Config?
Execution: Automatic attribution

NAM detects a configuration change on a network device, cross-references the timestamp against ROSS Active Directory login records, identifies the engineer responsible, and automatically emails them requesting their Change Order number for compliance.

📍
Locate Any Device Instantly
Execution: Seconds

An IP address is reported suspicious. NAM queries PaloAlto for the MAC, Cisco ARP tables for the switch, and CDP for the port — returning the exact physical location of the device across the entire network infrastructure in under 10 seconds.

🌐
ACL Push to All Sites
Execution: Minutes with validation

A new ACL rule needs to go to every branch site. NAM pushes it simultaneously to all Cisco devices, captures the output of each, validates the change was accepted, and emails a full result log — replacing what used to be a weekend change window with a 5-minute NAM job.
